# songs/index.json template

GitHub Pages cannot list folders automatically, so add each song here.

Example:

```json
[
  {
    "title": "AZiMA",
    "artist": "xi",
    "folder": "ANiMA",
    "chart": "chart.2d",
    "audio": "DeemoAnima.ogg",
    "updated": "2026-06-27"
  }
]
```

- folder: folder name under songs/
- chart: .2d file in that folder
- audio: audio file in that folder
- updated: used by updated-date sort
